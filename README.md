# Timeline Viewer

Локальное веб-приложение для просмотра событий на временной шкале и связанных медиафайлов.

## Назначение

- Загрузка папки с событиями.
- Отображение событий на таймлайне.
- Просмотр фото и видео, привязанных к выбранному событию.

## Формат данных

- Каждое событие — отдельная папка.
- Имя папки содержит дату или диапазон дат в квадратных скобках:
  - `[YYYY-MM-DD] Название`
  - `[YYYY-MM-DD/YYYY-MM-DD] Название`
- Дополнительные теги задаются квадратными скобками до блока даты:
  - `[tag1][tag2][YYYY-MM-DD] Название`

## Возможности

- Выбор папки через диалог или drag-and-drop.
- Фильтрация событий по тегам.
- Масштабирование таймлайна колесом мыши.
- Панель миниатюр с открытием в полноэкранном просмотрщике.
- Ручное изменение высоты блоков таймлайна и миниатюр через разделитель.

## Запуск

Статический клиент без сборки. Достаточно открыть `index.html` в браузере.
